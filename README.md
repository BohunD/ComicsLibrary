The app, which provides the user with information about Marvel characters, allows them to be added to a collection and allows users to write notes about each character in the collection.
Used tools: 
- Jetpack Compose
- Jetpack Navigation
- Coil
- MVVM
- Hilt
- Room Database
- Retrofit2
- Clean Architecture
